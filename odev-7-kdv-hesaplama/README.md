# Ödev 7 - KDV Tutarı Hesaplayan Program

## Açıklama
Bu program, kullanıcıdan alınan fiyat değerinin KDV'li fiyatını ve KDV tutarını hesaplayarak ekrana basar.

## KDV Oranları
- 0 - 1000 TL arası: %18 KDV
- 1000 TL üzeri: %8 KDV

## Örnek
- KDV'siz Fiyat: 10 TL
- KDV Oranı: %18
- KDV Tutarı: 1.80 TL
- KDV'li Fiyat: 11.80 TL

## Özellikler
- Kullanıcıdan klavye ile fiyat girişi (Scanner)
- Tutara göre otomatik KDV oranı belirleme
- KDV tutarı ve KDV'li fiyat hesaplama
