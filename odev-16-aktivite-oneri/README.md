# Ödev 16 - Aktivite Öneri Programı

## Açıklama
Bu program, girilen sıcaklık değerine göre aktivite önerir.

## Koşullar
- 5'den küçük → Kayak
- 5 ile 15 arası → Sinema
- 15 ile 25 arası → Piknik
- 25'ten büyük → Yüzme

## Farklı Çözüm Yolları
1. if-else if
2. Ternary Operator
3. Switch Expression
