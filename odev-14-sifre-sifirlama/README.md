# Ödev 14 - Şifre Sıfırlama Programı

## Açıklama
Bu program, kullanıcının şifresini kontrol eder ve yanlış şifre girilmesi durumunda şifre sıfırlama işlemi yapar.

## Akış
1. Kullanıcıdan şifre alınır
2. Şifre doğruysa "Giriş başarılı!" yazılır
3. Şifre yanlışsa kullanıcıya sıfırlamak isteyip istemediği sorulur
4. Sıfırlamak isterse yeni şifre alınır
5. Yeni şifre eskiyle aynıysa "Şifre oluşturulamadı, lütfen başka şifre giriniz." yazılır
6. Farklıysa "Şifre oluşturuldu." yazılır

## Özellikler
- String karşılaştırma (.equals())
- İç içe if-else kullanımı
- Scanner ile kullanıcı girişi
