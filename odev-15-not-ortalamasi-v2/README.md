# Ödev 15 - Not Ortalaması Hesaplayan Program V2

## Açıklama
Bu program, 5 dersin notlarını kullanıcıdan alarak ortalamasını hesaplar.
0-100 aralığı dışındaki notlar ortalamaya katılmaz.

## Özellikler
- Geçersiz notlar ortalamaya dahil edilmez
- Geçme notu: 55
- Ternary Operator ile geçti/kaldı kontrolü
