# Ödev 1 - Üniversite Sistemi

## Açıklama
Bu proje, bir üniversite sisteminin nesne yönelimli tasarımını içermektedir.

## Sınıflar
- **University** - Üniversiteyi temsil eder
- **Department** - Üniversiteye bağlı departmanları temsil eder
- **Office** - Çalışma ofislerini temsil eder
- **Classroom** - Sınıflıkları temsil eder
- **Employee** - Çalışanları temsil eden soyut sınıf
- **Professor** - Employee'dan türeyen profesör sınıfı
- **Officer** - Employee'dan türeyen memur sınıfı

## İlişkiler
- Üniversitenin sınıflıkları, ofisleri ve departmanları vardır
- Departmanların ofisleri vardır
- Çalışanlar (Profesör veya Memur) bir ofiste çalışır
